Privacy Policy

Last updated: April 23, 2025

1. Data Collection and Processing Principles

DeepShare is an open-source browser extension with user privacy protection at its core. We adhere to the following principles:

  • Zero Data Collection: We do not collect or store any of your personal information
  • Local Processing: All content processing is completed locally on your device
  • No Behavioral Tracking: We do not track your browsing behavior or use any analytics tools
  • No Third-Party Sharing: Your data is not shared with any third parties

2. Feature Privacy Explanation

2.1 Clipboard Data Processing

When using this tool:

  • DeepShare will only read clipboard content when you explicitly authorize it
  • Clipboard content is only used for format conversion
  • We will not store any of your clipboard data on any server

2.2 Conversation Screenshot and Formula Copy Features

All operations are performed locally in your browser:

  • The screenshot feature is executed only within your browser and does not involve data upload
  • Conversation content is only temporarily stored in browser memory and is not persistently stored
  • Exported images and text are saved directly to your device without passing through any third-party servers

2.3 Word Document Conversion Feature

When using the document conversion service:

  • We only process the content you select, and only for providing document conversion services
  • Data is immediately deleted after conversion is complete, and no copies are retained
  • Data transmission uses encryption to ensure security
  • We will never use your data for training models or other purposes

3. User Data Rights

As a DeepShare user, you have the following rights:

  • Transparency: Know how we process your data at any time
  • Not Being Tracked: Enjoy a tracking-free usage experience
  • Data Control: All data operations are controlled by you, and we will not access any data without your consent

4. Privacy Protection Measures

We take the following technical measures to protect your privacy:

  • Local Processing Priority: Process data locally on your device whenever possible
  • Minimum Data Principle: Process only the minimum data necessary to complete the function
  • Encrypted Transmission: Any necessary data transmission uses encryption
  • Timely Cleanup: Clear all data immediately after processing is complete

5. Privacy Policy Updates

If we make any changes to our privacy policy, we will post the updated version on this page and update the modification date.

6. Contact Us

If you have any questions or suggestions about our privacy policy, please contact us through GitHub Issues.
