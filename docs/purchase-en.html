Frequently Asked Questions

Why is the Word document conversion feature paid?
Most DeepShare features (like conversation screenshots, formula copying, etc.) are completely free. Only the Word document conversion feature requires an API Key purchase. This is because: 1) Document conversion requires server resources for high-quality formatting and processing; 2) A small fee helps prevent API abuse and ensures stable service for all users.
Does the API Key expire?
Yes, the API Key is valid for 1 year from the purchase date.
What if I forget my API Key?
If you've forgotten your API Key, please contact us via email with the email address you used for purchase, and we'll help you recover it.
How long until credits are added after payment?
After a successful payment, you will receive an email with your API key. The credits are added to your account immediately. If you don't receive the email, please check your spam folder and then contact us.
Didn't receive API Key after payment?
Please check your spam folder first. If you still can't find the email with your API Key, please contact us within 48 hours through any of the methods below!
