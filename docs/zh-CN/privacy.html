隐私政策

最后更新：2025-04-23

1. 数据收集与处理原则

DeepShare 是一个以用户隐私保护为核心的开源浏览器插件，我们坚持以下原则：

  • 零数据收集：我们不会收集或存储您的任何个人信息
  • 本地处理：所有内容处理均在您的设备本地完成
  • 无行为跟踪：我们不会跟踪您的浏览行为或使用任何分析工具
  • 无第三方共享：您的数据不会与任何第三方共享

2. 功能隐私说明

2.1 剪贴板数据处理

在使用本工具时：

  • 仅在您明确授权的情况下，DeepShare 才会读取剪贴板中的内容
  • 剪贴板内容仅用于格式转换
  • 我们不会存储您的任何剪贴板数据到任何服务器

2.2 对话截图和公式复制功能

所有操作均在您的浏览器本地进行：

  • 截图功能仅在您的浏览器内执行，不涉及数据上传
  • 对话内容仅临时保存在浏览器内存中，不会持久化存储
  • 导出的图片和文本直接保存到您的设备，不经过任何第三方服务器

2.3 Word 文档转换功能

在使用文档转换服务时：

  • 我们仅处理您选择的内容，且仅用于提供文档转换服务
  • 数据在转换完成后立即删除，不会保留任何副本
  • 数据传输采用加密方式，确保安全
  • 我们绝不会将您的数据用于训练模型或其他目的

3. 用户数据权利

作为 DeepShare 的用户，您拥有以下权利：

  • 透明了解：随时了解我们如何处理您的数据
  • 不被跟踪：享受无跟踪的使用体验
  • 数据控制：所有数据操作均由您控制，我们不会在未经您同意的情况下访问任何数据

4. 隐私保护措施

我们采取以下技术措施保护您的隐私：

  • 本地处理优先：尽可能在您的设备本地完成数据处理
  • 最小数据原则：仅处理完成功能所必需的最少数据
  • 加密传输：任何必要的数据传输均采用加密方式
  • 及时清除：处理完成后立即清除所有数据

5. 隐私政策更新

如果我们对隐私政策进行任何更改，我们将在此页面上发布更新的版本，并更新修改日期。

6. 联系我们

如果您对我们的隐私政策有任何疑问或建议，请通过 GitHub Issues 联系我们。
